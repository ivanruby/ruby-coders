Fase de configuración

Tarea: Establecer una asociación entre los componentes físicos en un circuito y sus representaciones en su programa.

Lo hará haciendo clic en un enlace a un proyecto Tinkercad y completando la actividad allí. Aquí está una tarea de calibración de ejemplo: Utilizando un LED con un botón - Parte 2 - Calibración

Enfoque: Programación, Pruebas

Lo que aprendemos: Cómo incluir, configurar y probar componentes físicos a través de la programación

Solución de ejemplo: Utilizando Arduino C/C++, configure los pines donde el LED y el botón están conectados

/* Paso 1. Crear una variable para el pin conectado al LED
           y otra para el pin conectado al botón 
   Consejo: Crear una variable le permite llamar a un pin por nombre
        en lugar de su número. */
byte LED = 2;
byte button = 3;

void setup()
{
  /* Paso 2. Configurar el pin LED como Salida y pin de botón como una Entrada

     Recordatoria: Cualquier pin programable (GPI) puede ser utilizado como
     un INPUT (recibir datos/voltaje) o OUTPUT (enviar datos/voltaje) */
     pinMode(LED, APAGADO);
     pinMode(botón, INPUT_PULLUP);
}
