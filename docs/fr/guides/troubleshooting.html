Dépannage

Dépannage, processus d'identification d'un problème, La planification et l'application d'une solution peuvent être réalisées pour suivre les phases que nous créons pour les projets (câblage, configuration et logique).

Voici des exemples de stratégies et de situations :

Wiring

  • Vérifie si le composant est câblé. Si ce n'est pas le cas, fermez-le

    Exemple : Oubliant de connecter un terminal d'une résistance, une LED, un bouton

  • Vérifiez si tous les composants du circuit sont câblés. Si ce n'est pas le cas, câblez-les

    Exemple : identique à l'exemple précédent

  • Vérifiez si le câblage est correct pour le composant. Si ce n'est pas le cas, filer correctement

    Exemple : Inverser le câblage sur une LED en connectant le terminal positif à une broche au sol (GND)

  • Vérifiez si le composant est alimenté correctement. Si ce n'est pas le cas, faites-le correctement

    Exemple : Câblage d'une composante à 3.3 Volts lorsqu'il faut 5 Volts

Configuration

  • Vérifiez si vous avez créé une variable pour la broche connectée au composant
  • Vérifiez si vous avez configuré la broche avec le mode approprié
  • Vérifiez si vous avez initialisé le composant, s'il en a besoin

Logique

  • Vérifiez si vous avez créé une variable pour la broche connectée au composant
  • Vérifiez si vous avez configuré la broche avec le mode approprié
  • Vérifiez si vous avez initialisé le composant, s'il en a besoin
