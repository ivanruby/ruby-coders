Como tudo começa

No início das nossas actividades, temos uma descrição do projecto. Tanto quanto possível, os projetos em que trabalhamos são concebidos para integrar cenários do mundo real. No entanto, antes de lá chegar, precisamos também de desenvolver uma compreensão da forma como as diferentes componentes que podemos utilizar funcionam.

Nesses casos, os projectos em que trabalhamos podem parecer menos representativos de um cenário real, mas são essenciais para compreender como as coisas funcionam.

Prompt

Como exemplo, vamos considerar a seguinte descrição:

Neste projeto, um LED deve ligar (iluminar) quando um botão é pressionado.
Ambos os componentes estão conectados a um microcontrolador Arduino.

Nota: A descrição parece desafiadora ou inclui conceitos estrangeiros para você? Não se preocupe. Nós adotamos uma abordagem passo a passo e trabalhamos em explorar conceitos, porque precisamos eles, em vez de simplesmente memorizar
