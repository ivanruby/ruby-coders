Piscar LED com um botão

Neste projeto, dois LEDs devem piscar alternando entre eles
uma vez que um botão é pressionado. Caso contrário, ambos os LEDs devem permanecer ON.

Estados

  • Botão não pressionado: LED A, B ON
  • Pressionado botão: LED A LIGADO, LED B DESLIGADO por 1 segundo e então o oposto
