O Arduino

O quadro Arduino é um microcontrolador popular e de código aberto usado em uma variedade de projetos. Ele se comporta como um pequeno computador e pode executar tarefas como ler dados dos sensores, executar operações e decisões com base nestes dados e enviar dados para outros dispositivos.

Você pode ler mais sobre isso no guia oficial do Arduino.

Ele inclui um conjunto de pinos que podem ser conectados, ou com fio, para outros componentes. Estes pins podem ser classificados em programáveis e não programáveis.

Pinos não programáveis

Estes são pinos que têm um comportamento fixo. Os pins não programáveis mais usados são aqueles que usaremos para dar poder (voltagem) a nossos circuitos. Estes pins serão positivos (3,3 Voltas ou 5 Tensão ) e negativos (Ground - GND).

Fixos programáveis

Usamos pins programáveis sempre que queremos atribuir comportamentos específicos a componentes em nossos circuitos. Vamos controlar estes pins utilizando nosso programa e, no tabuleiro Arduino Uno, eles são divididos em Analógico (A0 para A5) e Digital (0 a 13) pinos.
