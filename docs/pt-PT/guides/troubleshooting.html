Solução de problemas

Solucionando, o processo de identificação de um problema, planear e aplicar uma solução para ela pode ser feito quanto a seguir as fases que criamos para os projetos (pareamento, configuração e lógica).

Veja as estratégias amostrais e situações:

Wiring

  • Verifique se o componente está com fio. Se não, arame o telefone

    Exemplo: Esqueça de conectar um terminal de resistor, um LED, um botão

  • Verifique se todos os componentes no circuito estão conectados. Se não, transfira-os

    Exemplo: Igual ao exemplo anterior

  • Verificar se o cabeamento está correto para o componente. Se não, arame corretamente

    Exemplo: Revertendo a fiação em um LED conectando o terminal positivo a um pin

  • Verifique se o componente é fortalecido apropriadamente. Se não, energize-o adequadamente

    Exemplo: Conectando um componente com 3.3 Volts quando requer 5 Volts

Configuração

  • Verifique se você criou uma variável para o pin que está conectado ao componente
  • Verifique se você configurou o pino com o modo apropriado
  • Verifique se você inicializou o componente, se ele precisar

Lógica

  • Verifique se você criou uma variável para o pin que está conectado ao componente
  • Verifique se você configurou o pino com o modo apropriado
  • Verifique se você inicializou o componente, se ele precisar
